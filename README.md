# 📊 Zomato Business Intelligence & Data Analysis Project

This project provides a comprehensive analysis of Zomato's restaurant data across various cities in India. It combines the power of **SQL** for data querying and **Power BI** for interactive visualization to uncover key business insights.

## 🚀 Project Overview
The goal of this project is to analyze restaurant distribution, pricing strategies, and customer ratings to help understand market trends in the food-delivery industry.

## 🛠️ Tools & Technologies Used
- **Power BI:** For creating an interactive dashboard with Map visuals, KPIs, and Slicers.
- **SQL (SQLite):** For data extraction, filtering, and performing aggregations.
- **Excel:** For raw data storage and cleaning.

## 📈 Key Insights from Dashboard
- **Geographic Reach:** Analyzed presence across 17+ cities using interactive Map visuals.
- **Top Cuisines:** Identified the most popular food categories (Beverages, Pizza, and North Indian).
- **Pricing Analysis:** Visualized the average cost for two across different restaurant categories.
- **Quality Metrics:** Tracked average ratings to identify top-performing restaurant hubs.
